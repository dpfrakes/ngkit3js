Slideshow - Hero Carousel

Notes:
Use Proper JSON notation when setting data attributes: 
    
        <div class="ng-slidenav-position ng-hero-carousel"
             data-ng-slideshow='{"autoplay":true,"autoplay-interval":3000, "animation":"fade", "view-height":80}' >
Also on the ul.ng-slideshow element, we need to include the class ng-slideshow-items as well so the JS can detect the list: 
    
        <ul class="ng-slideshow ng-slideshow-items" style="margin-left: auto; margin-right: auto; width: 100%; padding-left: 0 !important;">
